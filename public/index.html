HOUSEKEEPING

Stale Slot Cleanup

Harbor only targets isolated `codex-*` homes that still have local sessions but never completed login, so they remain safe to archive as stale slots.

RECOVERY

Isolation Repair

This detaches any `sessions` symlink that still points at a shared folder and restores local copies per account.

Use Restore Shared History if you want to re-import the old symlink-era chat history from the configured shared sessions root without re-sharing auth state.

Saved target presets let you reuse common restore groups such as main only, all codex slots, or one specific account.

RESTORE POINTS

Backup Catalog

Harbor only catalogs its own restore points: session index snapshots, state DB reset backups, archived isolated slots, auth restore points, and launcher backups.

TRACE

Audit Log

Harbor keeps a local append-only audit trail for operational changes such as repair, restore, cleanup, launcher writes, preset edits, and Health auto-fixes.

TRANSFER

Session Shuttle

This uses the latest session found in the selected source home's `history.jsonl`. If the live thread has not flushed into `sessions/` yet, Harbor materializes a portable session first and then broadcasts it safely.
